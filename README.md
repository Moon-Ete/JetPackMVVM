# JetPackMVVM

使用Google Jetpack组件库搭建的MVVM架构

依赖引入方式

1. 添加repositories

   ```
   repositories {
   		...
           maven { url 'https://www.jitpack.io' }
           ...
       }
   ```



2. 添加dependencies 依赖

   ```
   dependencies {
   		implementation 'com.github.Moon-Ete:JetPackMVVM:lastVerison'
   	}
   ```
